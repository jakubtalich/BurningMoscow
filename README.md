# BurningMoscow
Custom Advent of Code 2025 project displaying burning Saint Basil's Cathedral with music playback

// Wishing you a Christmas so bright that even state propaganda can't extinguish it. //

## Compiling

Dependencies: mpg123, ffplay or play

```bash
dart compile exe moscow.dart
```

## Usage

```bash
moscow <path_to_mp3_file>
```
